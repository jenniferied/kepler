# Kepler

Kepler's website -- music, visuals, digital identity.

## What This Is
Static website (GitHub Pages). Vanilla JS, ES6 modules, EventBus architecture.

## Commands
```bash
python3 -m http.server 8001              # Website dev
npm run build:css                         # Build Tailwind CSS
npm run watch:css                         # Watch mode (dev)
node scripts/generate-journal-manifest.js # Update journal
```
